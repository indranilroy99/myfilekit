# MyFileKit

MyFileKit is an all-in-one file and business tools workspace for invoices, PDFs, images, signatures, and everyday document workflows.

## Current Tools

- Premium invoice generator with templates, custom fields, payment details, TDS, signatures, logo controls, and print/PDF export.
- Image compressor that runs in the browser using canvas.
- PDF merge and split tools that run in the browser with `pdf-lib`.
- Tool dashboard covering the broader PDF, image, business, security, and AI document roadmap.

## Local Usage

Open `index.html` in a browser for the MyFileKit dashboard, or open `invoice-generator/index.html` directly for the invoice tool.

## Notes

This project is intentionally client-first where possible, so files can be processed locally in the browser before any hosted backend exists.
